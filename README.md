## What is MiOffice?

[**MiOffice**](https://mioffice.ai) is a free, browser-based file processing platform with **66+ tools** for PDF, image, video, and AI-powered operations. Everything runs **100% client-side** using WebAssembly ‚Äî your files never leave your device.

No uploads. No servers. No subscriptions. Just tools that work.

## Why MiOffice?

### üîí Privacy First
Your files are processed **entirely in your browser**. Nothing is uploaded to any server. We can't see your files even if we wanted to ‚Äî because they never leave your device.

### ‚ö° WebAssembly Powered
[MiOffice](https://mioffice.ai) uses cutting-edge **WebAssembly** technology for near-native processing speed directly in your browser. No software installation required.

### üÜì Free & Unlimited
No file size limits. No daily usage caps. No watermarks. No accounts required. Just open [mioffice.ai](https://mioffice.ai) and start processing.

### üåç Works Everywhere
Runs in any modern browser ‚Äî Chrome, Firefox, Safari, Edge. Desktop or mobile. No plugins or extensions needed.

### üöÄ Lightning Fast
Client-side processing means no upload/download wait times. Your files are processed instantly using your device's computing power.

## Tech Stack

[MiOffice](https://mioffice.ai) is built with modern web technologies:

- **[Next.js](https://nextjs.org/)** ‚Äî React framework with SSR and static generation
- **[TypeScript](https://www.typescriptlang.org/)** ‚Äî Type-safe development
- **[WebAssembly](https://webassembly.org/)** ‚Äî Near-native performance in the browser
- **[Web Workers](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API)** ‚Äî Non-blocking background processing
- **[ONNX Runtime](https://onnxruntime.ai/)** ‚Äî AI model inference in the browser
- **[pdf-lib](https://pdf-lib.js.org/)** ‚Äî PDF manipulation
- **[Tailwind CSS](https://tailwindcss.com/)** ‚Äî Utility-first styling
